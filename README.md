# PRODIGY_FS_02
🛒 Local Store E-Commerce Platform built with HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB. Features product listings, shopping cart, secure checkout, responsive design, and an easy-to-use interface. Designed to help local businesses sell products online with a seamless shopping experience.

# 🛒 Local Store E-Commerce Platform

A modern and responsive **Local Store E-Commerce Platform** developed to help local businesses sell their products online. This platform allows customers to browse products, view detailed information, add items to the shopping cart, and place orders through a simple and user-friendly interface.

The project is designed with a clean layout, responsive design, and essential e-commerce features to provide a smooth shopping experience for users while making product management easier for store owners.

# 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
4. Browse products, add them to the cart, and complete the checkout process.
