# рҹӣ’ Local Store E-Commerce Platform

A modern and responsive **Local Store E-Commerce Platform** developed to help local businesses sell their products online. This platform allows customers to browse products, view detailed information, add items to the shopping cart, and place orders through a simple and user-friendly interface.

The project is designed with a clean layout, responsive design, and essential e-commerce features to provide a smooth shopping experience for users while making product management easier for store owners.
